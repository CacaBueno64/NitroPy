# NitroPy

This is a python library for supported Nitro formats found in nds games :
- Nsbmd (3D models, with optionnal textures/palettes) : ✔️
- Nsbtx (Textures) : ➖
- Nsbca (Joint animations) : ⏳
- Nsbta (Texture animations) : ❌
- Nsbtp (Pattern animations) : ❌
- Nsbma (Material animations) : ❌
